Scania UI Guidelines

This is Scania UI Guidelines

How do you create a great user experience (UX)? Start here!

Here we present a graphic design with colour schemes and fonts together with design guidelines for common design elements. You will soon be able to find guidance on such things as error messages, buttons, tables etc. By using this you help us create a common user experience for the Scania end users. They will be able to recognize their internal application as a Scania application. They will be able interact with common design elements in similar ways regardless of the application. Thereby we will help them focus on their work instead of the IT system.

To those of you who develop web applications, we also offer you Scania Bootstrap. Here you can find descriptions of how you can develop with the Scania Bootstrap CSS and JS-framework. You can find examples, codes for each component and functional prototypes and examples.

This is only a start package. If you want to take UX (and usability) seriously, you should also involve UX resources in your project. If you want technical assistance, contact Jesper Karlsson. If you want general guidance on UX or usability, contact Susanna Christensson. We are happy to meet up and discuss possible ways that we, or others in the UX team can help you.

Building a new application?

Did you know that we're currently working on a totally new user experience? Scania Boostrap is currently extended to form the new Corporate UI. It includes our new branding guidelines including fonts, logotype and colors.

If you're using Scania Bootstrap today, the basic updates that you'll need to do will be available in version 2.0.0. If you're not using Scania Bootstrap, start planning to do so! Find out more about the new interface at the UX Library

Up and running quickly

Re-use our ready-made UI-components

There is no need inventing that what already exists! Our extensive library of component and JS-functionality is a goldmine when designing an application. Use this together with our UI-guidelines and you get a lot of hard UX-work for free.

Fast responsetimes, 100% uptime

Online CDN hosting

With our CSS, JS, Images and font-files stored in data-centers all over the world, we can ensure the lowest possible reponse-times for our files. And when the user is moving between different Scania Bootstrap-applications, the experience is much quicker since the common files are cached locally.
