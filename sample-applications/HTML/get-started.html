Introduction

Get started with Corporate UI 4.0.0. This is a early release that's focused mainly on a solid architecture and making things work in Google Chrome.

Quick start

Looking to quickly add Corporate UI to your project? Use our CloudFront CDN. Want to use a package manager? Available as NPM package.

Corporate UI CDN Service

Copy-paste the Corporate UI initializer <script> just before the </head> tag.

<script src="https://static.scania.com/build/global/4.x/corporate-ui.js"></script>

Change the version number, You can find available versions here

Styling of Custom Elements

All styling of custom elements are done in the build phase and you'll define what theme to apply using the c-theme component. These styles are scoped within each component and will not leak out to other parts of your document. You can set global attribute to be able to use Bootstrap classes and javascript. 

  <c-theme name="scania" global="true"></c-theme>
