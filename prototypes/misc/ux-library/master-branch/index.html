The Master

In the Master Branch we're continuously improving the Corporate UI. At this site, you can preview and test pull requests made at the Corporate UI GitHub repository. The whole site are created for testing purposes only, so expect some outdated information here and there. Feel free to add your own sample pages by making a pull request in the Docs Repository. Scania Bootstrap is the HTML/CSS framework for implementing web applications at Scania. Basically, you can use all Bootstrap Classes and Components.
