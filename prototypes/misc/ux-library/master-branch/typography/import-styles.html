Scania Bootstrap - a real money saver!

Scania Bootstrap is based on the very well established CSS framework Bootstrap, founded by Twitter. Thousands of hours are already spent developing the basic framework and we're just giving it an unique touch of Scania. Find out more about the original Bootstrap at Wikipedia.

Why spending a few hundred hours on re-inventing the wheel when we can help you set up a phenomenal foundation for a fraction of the cost?

Totally responsive - for free!

Using the Scania Bootstrap CSS framework, you'll have all the responsive design support that you'll ever need. Try simulate a smartphone or tablet simply by re-sizing this browser window.

As you can see, all columns are re-adjusted automatically without any effort what so ever. The horizontal menu is transformed into a drop down menu so that the end-user will be able to navigate without scrolling or swiping on the screen.

Sample code for creating a modal overlay

<div class="modal fade" id="modal-x" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <button type="button" class="close" data-dismiss="modal" aria-hidden="true">&times;</button> <h4 class="modal-title">Modal title</h4> </div> <div class="modal-body"> <p>One fine body&hellip;</p> </div> <div class="modal-footer"> <button type="button" class="btn btn-primary btn-sm">Save</button> <button type="button" class="btn btn-default btn-sm" data-dismiss="modal">Close</button> </div> </div><!-- /.modal-content --> </div><!-- /.modal-dialog --> </div><!-- /.modal -->

And the button to toggle the modal

<button class="btn btn-primary btn-sm" data-toggle="modal" data-target="#modal-x">Show modal overlay</button>
