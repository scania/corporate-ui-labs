Logotype

For print usage, the Scania logotype (symbol and wordmark) is supplied by Scania Media Provider. For web application development, the symbol and wordmark are included among our ready to run web components.

For Web Usage

For use in applications where we can't use either Polymer web components or the Scania Bootstrap framework. The Scania logotype stylesheet includes both scalable SVG versions as well as PNG fallback versions and are already included and available in the Corporate UI.

Social Media

If you are running a Scania channel in social media and need the Scania Symbol for the profile page, feel free to download the Scania social media symbols. There are a few versions available including Facebook, Instragram, Twitter, LinkedIn and YouTube.

Need help?

If you have any questions regarding of how the symbol and wordmark should be used on the web, please send us an e-mail and we'll be happy to help out.
