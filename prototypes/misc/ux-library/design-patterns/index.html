Corporate UI Design Patterns

This design patterns library covers basic styles, individual elements, and more complex patterns. They define how interactions behave, what they look like, and how they might be implemented. The main idea is that you'll use our more complex Web Components for solving general, cross functional tasks, while the patterns found here should be used to develop the key functionality of you application.

By using the components and elements in this library you'll help us create a seamless user experience for the Scania end users. They will be able to recognize their internal application as a Scania application. They will be able interact with common design elements in similar ways regardless of the application. Thereby we will help them focus on their work instead of the IT system.

This is only a start package. If you want to take UX (and usability) seriously, you should also involve UX resources in your project. If you want technical assistance, contact Jesper Karlsson.
