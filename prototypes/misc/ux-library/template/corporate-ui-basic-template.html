Alpha version of the all new Corporate UI.

PLAY AROUND!

This template is running on a Beta version of the new Web Component based framework called Corporate UI. To run this file at your server, you'll need sure that CORS is enabled.

Please use this template and change content and parameters to fit your needs. Use additional components from the UX Library. Stay in contact with the developers using corporate-ui@scania.com
